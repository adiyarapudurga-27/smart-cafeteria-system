# Smart Cafeteria System

The Smart Cafeteria System is a modern web-based application developed to simplify and improve cafeteria management and food ordering processes. The application provides a user-friendly interface for customers to browse food items, place orders, manage favorites, and track their orders efficiently. It also includes an admin dashboard for managing menu items, orders, and cafeteria operations.

This project is designed to reduce waiting time, improve customer experience, and digitize traditional cafeteria services using modern web technologies.

## Key Features

### User Features

* User Registration and Login
* Browse Food Menu
* Add Items to Cart
* Favorite Food Items
* Place Orders Easily
* Order Success Tracking
* Responsive and Attractive User Interface

### Admin Features

* Admin Dashboard
* Menu Management
* Order Management
* Customer Management
* Analytics and Reports
* Table and Cafeteria Management

## Technologies Used

* React.js
* TypeScript
* Vite
* Tailwind CSS
* Context API
* Modern UI Components

## Project Objective

The main objective of this project is to provide a smart and efficient cafeteria management solution that improves order handling, reduces manual work, and enhances the overall dining experience for users.
